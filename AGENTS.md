This project is a rewrite of the MView software into a modular frontend/backend architecture. The backend is implemented in Python. The frontend supports both a web-based interface and a command-line interface. The following Markdown design documents are important:

- design/myview-analysis.md contains the description of the MView software (source code in `baseline/`). Refer to this file in priority over the actual source code.
- design/pyview-design.md contains the overall design goals and architecture. Always adhere to this.
- design/work-plan.md contains a week-by-week breakdown of the tasks. Follow this plan as you create each component.
- design/tech-stack.md contains the minimal list of technologies you must adhere to. Any extra dependencies must be proposed and justified by the manager.
